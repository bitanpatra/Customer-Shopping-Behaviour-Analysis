# Customer Shopping Behavior Analysis

This project is an end-to-end data analytics case study analyzing customer shopping behavior using transactional data.  
The analysis focuses on identifying spending patterns, customer segments, subscription behavior, and actionable business insights.

**Tools used:** Python, SQL Server, Power BI

---

## Project Overview
The project follows a complete analytics workflow:
- Data cleaning and feature engineering using Python
- Business-focused analysis using SQL Server
- Interactive reporting and visualization using Power BI
- Translation of analytical insights into business recommendations

A detailed analytical report and presentation are included for full context.

---

## How to Run the Project

### 1. Clone the Repository
```bash
git clone https://github.com/bitanpatra/customer-shopping-behavior-analysis.git
cd customer-shopping-behavior-analysis
```

### 2. Python Analysis

Open the Jupyter Notebook in the python/ directory

Install required libraries if not already available:

```bash
pip install pandas numpy matplotlib seaborn
```


Run the notebook cells in order to perform data cleaning and feature engineering

### 3. SQL Analysis

Import the cleaned CSV file into Microsoft SQL Server

Execute the SQL scripts in the sql/ directory to reproduce the analysis

### 4. Power BI Dashboard

Open the .pbix file in Power BI Desktop

Refresh data connections if required

Explore interactive visuals and slicers to review insights
